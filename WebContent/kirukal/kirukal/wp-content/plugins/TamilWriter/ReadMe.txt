Tamil Writer 1.1 by Monusoft
http://www.monusoft.com/Products/WordPressPlugin/tabid/187/Default.aspx

Easily write post/page/comment in Tamil with easy-to-use Tamil transliterate keyboard. No need of Tamil IME. 

--------------------------------------------------------------

Features:

    * Easily write post/page/comment in Tamil.
    * No need to learn Tamil typing.
    * Easy inline editing of post in Tamil.
    * Switch to normal english keyboard by pressing F12 or selecting "English" radio button.
    * Extremely fast Tamil typing.
    * It's all magic of Javascript.

Plugin Installation:

   1. Unzip/upload to /plugins directory.
   2. Plug-in file path will become "wp-content/plugins/TamilWriter/TamilWriter.php"
   3. Activate via WordPress Plugins tab.
